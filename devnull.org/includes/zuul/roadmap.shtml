Public/Private Key Encryption

Allow properties to be encrypted asymmetrically with PGP public and private keys. This provides a more secure mechanism to keep your sensitive data safe.

Stronger PBE Encryption

Currently, the PBEWithMD5AndDES algorithm is used to encrypt values. Future versions will support PBEWithMD5AndTripleDES but will require Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files installed.

Environment Based Roles

Enable roles based on environment. For instance, allow your development staff full control over dev environments but restrict the production environments to your operational staff.

Hierarchical Configurations

Mash up multiple configurations into a single configuration with inheritance and overridden values.

Syndicated Feeds

Subscribe to changes via RSS or Atom feeds.

Dependency Analysis

Track download activity in order to gain oversight into what configuration settings are being used.
